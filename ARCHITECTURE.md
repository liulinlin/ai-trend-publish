## 数据流转

### 方案1：miyucaicai（简单方案）

```
用户点击刷新
    │
    ▼
trend-manager.js
    │
    │ callFunction('hotspot-miyucaicai')
    ▼
云函数: hotspot-miyucaicai
    │
    │ axios.get('https://top.miyucaicai.cn/')
    ▼
miyucaicai网站
    │
    │ HTML响应
    ▼
云函数: 解析HTML
    │
    │ 正则匹配热点数据
    ▼
返回标准化数据
    │
    ▼
前端展示热点列表
```

### 方案2：MediaCrawler（完整方案）

```
用户点击刷新
    │
    ▼
trend-manager.js
    │
    │ callFunction('mediacrawler-hotspot', {platform: 'xhs'})
    ▼
云函数: mediacrawler-hotspot
    │
    │ POST /api/crawler/start
    ▼
MediaCrawler API
    │
    │ 启动爬虫任务
    ▼
Playwright浏览器
    │
    │ 访问小红书
    ▼
小红书平台
    │
    │ 返回搜索结果
    ▼
MediaCrawler解析
    │
    │ 保存为JSON
    ▼
云函数轮询状态
    │
    │ GET /api/crawler/status
    ▼
爬虫完成
    │
    │ GET /api/data/list
    ▼
获取爬取数据
    │
    │ 转换为标准格式
    ▼
返回给前端
    │
    ▼
前端展示热点列表
```

## 云函数对比

| 特性 | mediacrawler-hotspot | hotspot-miyucaicai | hotspot-collector |
|------|---------------------|-------------------|-------------------|
| **数据源** | 7个平台 | 聚合网站 | V2EX/微博/知乎 |
| **数据质量** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| **响应速度** | 慢(10-30s) | 快(2s) | 中(5s) |
| **稳定性** | 中 | 高 | 高 |
| **配置复杂度** | 高 | 低 | 低 |
| **依赖** | MediaCrawler服务 | 无 | 无 |
| **推荐场景** | 生产环境 | 开发/降级 | 补充数据 |
