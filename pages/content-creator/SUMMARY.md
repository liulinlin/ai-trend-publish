## 🎨 核心功能特性

### 1. 三步创作流程

```
步骤1: 选择热点
  ├─ 实时热点展示
  ├─ 分类筛选
  ├─ 热度排序
  └─ 一键选择

步骤2: 内容创作
  ├─ 创作类型选择
  ├─ 参数配置
  │   ├─ 内容风格（5种）
  │   ├─ 内容长度（4种）
  │   └─ 目标平台（5种）
  └─ 补充需求输入

步骤3: 优化发布
  ├─ 内容展示
  ├─ 优化选项
  │   ├─ 格式优化
  │   ├─ SEO优化
  │   └─ 智能配图
  └─ 多平台发布
      ├─ 微信公众号
      ├─ 小红书
      ├─ 知乎
      ├─ 抖音
      └─ B站
```

### 2. AI模型集成

**模型配置**：
- 模型：glm-4.7-flash
- 提供商：智谱AI
- 特点：快速响应、高质量输出

**提示词工程**：
```javascript
buildPrompt() {
  // 动态构建提示词
  // 包含：任务描述、热点主题、创作要求、输出规范
  return prompt;
}
```

### 3. 数据流转

```
热点数据流：
云函数(hotspot-collector) 
  → 本地缓存(30分钟) 
  → 页面展示 
  → 用户选择 
  → 传递到创作页面

内容生成流：
用户配置 
  → 构建提示词 
  → 调用GLM API 
  → 内容生成 
  → 展示结果 
  → 保存/发布
```

## 📊 功能对比

### 原有功能（视频创作）

```
热点选择 → 视频脚本 → 分镜生成 → 图片生成 → 视频合成
```

### 新增功能（自媒体创作）

```
热点选择 → 内容生成 → 内容优化 → 多平台发布
```

### 共同点

- 都基于热点数据
- 都使用AI生成
- 都有多步流程
- 都支持参数配置

### 差异点

| 维度 | 视频创作 | 自媒体创作 |
|------|---------|-----------|
| 输出类型 | 视频 | 文本内容 |
| 生成时长 | 较长（分镜+图片） | 较短（纯文本） |
| 平台适配 | 短视频平台 | 图文平台 |
| 后续处理 | 视频编辑 | 格式优化 |
